# slice 切片

## 声明
切片相当于可扩容数组，可以直接声明
```go
s := []int{}
```
或者使用`make()`函数构造
```go
//make([]T, size, cap)
s := make([]int, 2, 10)
```
## 切片判空
```go
len(s) == 0
```

###切片删除元素
Go的slice没有删除方法，可以利用自身append方法达到目的
```go
s := []int{1, 2, 3, 4, 5, 6, 7}
fmt.Println(s) //输出[1 2 3 4 5 6 7]
//删除第二个元素
s = append(s[:1], s[2:]...)
fmt.Println(s) //输出[1 3 4 5 6 7]
```
